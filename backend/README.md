# StudyRoom Backend - Firebase Connection

Minimal Firebase connection setup.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Firebase is already configured in `index.js`

3. Use the connection:
```javascript
const { db } = require('./index.js');
// Now you can use db to interact with Firestore
```

## Firebase Config

The Firebase configuration is set up for project: `ui-design-studyroom`
